Here are 5 HTML tags I've learnt and their descriptions.

Tag p /p:
This one is used for adding a parapgraph.
Tag em /em:
This one is used for emphatized text. When we use this tag the text is displayed in italic.
Tag sup /sup:
This one is used for superscript text, like ordinal numbers (3o).
Tag blockquote /blockquote:
This one is used to cite a quote from another source.
Tag strong /strong:
This one is another way to emphatized an important text. When we use this tag the text is displayed in bold.
